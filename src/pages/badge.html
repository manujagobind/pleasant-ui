Badge on Avatars

To add a status badge to an avatar, use a has-badge class along with the avatar. Within it add a span element with the status badge of your choice. The available choices are badge-online, badge-offline, badge-idle, and badge-busy. 

                    
    <div class="avatar-xl has-badge">
        <img src="https://picsum.photos/300/300" alt="Sample avatar" class="img-responsive img-round">
        <span class="badge badge-online"></span>
    </div>
    <div class="avatar-lg has-badge">
        <img src="https://picsum.photos/300/300" alt="Sample avatar" class="img-responsive img-round">
        <span class="badge badge-offline"></span>
    </div>
    <div class="avatar-md has-badge">
        <img src="https://picsum.photos/300/300" alt="Sample avatar" class="img-responsive img-round">
        <span class="badge badge-idle"></span>
    </div>
    <div class="avatar-sm has-badge">
        <img src="https://picsum.photos/300/300" alt="Sample avatar" class="img-responsive img-round">
        <span class="badge badge-busy"></span>
    </div>

Badge on Icons

To add a badge on icons, use the has-badge class along with the icon and within it, add a span element with class badge and the badge content as text.

30 
                    
    <i class="fas fa-heart has-badge">
        <span class="badge">30</span>
    </i>
