Badge Component

The following CSS import is required to use the badge component.

                    
    @import url("https://pleasant-ui.netlify.app/src/styles/badge.css");
    @import url("https://pleasant-ui.netlify.app/src/styles/avatar.css");
    @import url("https://pleasant-ui.netlify.app/src/styles/image.css");
